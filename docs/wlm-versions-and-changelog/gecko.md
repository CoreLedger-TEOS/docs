# Gecko

<p className="doc-description">Gecko version coming soon at 30th June.</p>

### Changes:

* This release introduces several enhancements to CoreLedger's Internet Computer integration, refining the underlying implementation and improving the overall user experience when operating on the Internet Computer network. In addition, we have re-established a unified code base across all blockchain infrastructures supported by CoreLedger TEOS, ensuring long-term consistency and providing a clear migration path between supported blockchains.
* Update of Auth Server libraries
* Added support for EIP-1559 on Polygon, introducing dynamic fee transactions with base fee and priority fee mechanisms.
