# Settings

Settings represent the global configuration parameters of the TEOS system that define how supplies and controllers operate.\
They include limits, thresholds, and operational rules that affect system behavior, such as supply expiration policies and transaction address limits.\
These settings are read-only and can be retrieved to understand the current system configuration.

📘 Learn more about [supplies](../../resources/using-the-teos-api/concepts/supply.md) and [controllers](../../resources/using-the-teos-api/concepts/controllers.md).
