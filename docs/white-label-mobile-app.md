# Welcome!

WLA version 0.23 (released in March, 2024)

:::info
New version v0.23 is available. Check what's new in the WLA versions and changelog.
:::

White Label Mobile App (WLA) is a customized mobile app which acts as blockchain wallet within TEOS and can be used for signing transactions, checking balances, creating and paying invoices, creating supplies and running trades.

:::info
TEOS platform is CoreLedger's main product. It allows companies to integrate blockchain technology into their business processes.
:::

While working with WLA, since it is part of TEOS platform, you may also need to check the docs of other TEOS platform components:

- [White Label Management Tool](./white-label-management-tool.md)
- [TEOS API](./index.md)
- [White Label Portal](./white-label-portal.md)

Please be aware that the following figures, assets and names might be different from your customized app.
