# Admin user guide

This section describes how admins configure and prepare White Label Mobile App (WLA).

We recommend admins to start with following steps:

1. Check which modules can be included in the app: [WLA modules](./admin-user-guide/wla-modules.md)
2. Complete login and wallet setup: [Login and wallet creation](./admin-user-guide/login-and-wallet-creation.md)
3. Adjust the app configuration to your needs: [App configuration](./admin-user-guide/app-configuration/index.md)
4. Check how your end users interact with the app: [How your users interact with WLA](./how-your-users-interact-with-wla/index.md)
